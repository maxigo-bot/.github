# maxigo-bot

Go ecosystem for building bots on [Max Messenger](https://max.ru).

| Package | Description | Status |
|---------|-------------|--------|
| [maxigo-client](https://github.com/maxigo-bot/maxigo-client) | Idiomatic HTTP client for Max Bot API | v0.1.0 |
| maxigo | Bot framework with router, middleware, and context | In development |
| maxigo-proxy | Telegram → Max proxy — run existing Telegram bots on Max | Planned |
